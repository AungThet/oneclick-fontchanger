Usage for oneclick-fontchanger

You can use the keyboard shortcut Ctrl+B to change chrome's font settings to first font and Alt+B to change to second font.

Don't like default shortcuts? Lucky for you! Chrome has built-in support for customizing shortcuts for extensions.

  • Please go to extensions page.
  • You will see a link lower right corner of the page named "keyboard shortcuts".
  • Click on it.
  • Then find the oneclick-fontchanger and change the shortcuts.
