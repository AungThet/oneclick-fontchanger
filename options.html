Usage for oneclick-fontchanger

You can use the keyboard shortcut Ctrl+Shift+1 to change chrome's font settings to first font and Ctrl+Shift+2 to change to second font.

*** You can remap the key via Keboard Shortcuts button just below chrome's extensions page

Settings

Font Name for First Shortcut(Default Ctrl+Shift+ 1)
Font Name for Second Shortcut(Default Ctrl+Shift+ 1)
